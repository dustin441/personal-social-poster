# Testing Checklist

Before publishing a real post:

- [ ] Google Sheet exists
- [ ] Required columns exist
- [ ] n8n can write a row
- [ ] local script can read the Sheet
- [ ] browser opens correctly
- [ ] user is logged into the platform
- [ ] test row has safe content
- [ ] only one row is marked `ready_to_post`
- [ ] dry run succeeds
- [ ] user approves real test post
- [ ] script updates status after posting
- [ ] duplicate prevention works

Suggested real test post:

`Testing a new posting workflow. If you see this, the setup worked.`
