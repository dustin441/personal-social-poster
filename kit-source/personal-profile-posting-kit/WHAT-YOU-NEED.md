# What You Need

You need:

- A Google account
- A Google Sheet
- n8n Cloud or self-hosted n8n
- A Mac or Windows computer that can stay on
- Chrome or Chromium installed
- Node.js 20+
- Your personal social account already logged into the browser
- Optional: Hermes, OpenClaw, Claude Code, Codex, or another coding agent

Recommended computer setup:

- Best: a Mac mini, old laptop, desktop, or spare always-on computer
- Acceptable: your regular computer if it is awake during posting windows
- Advanced/riskier: remote desktop or cloud browser

Why local?

Personal social platforms trust your normal browser, device, cookies, and location. Running from your own computer is usually more reliable than running from a random cloud server.
