# Go-Live Checklist

Before enabling automatic polling:

- [ ] One real post was published successfully
- [ ] Sheet was updated after posting
- [ ] error logging works
- [ ] screenshot capture works on failure
- [ ] duplicate prevention works
- [ ] computer sleep settings are adjusted
- [ ] scheduled task/LaunchAgent is configured
- [ ] user knows how to pause automation

Pause automation:

- Set `local_status = paused`, or
- Disable the scheduled task/LaunchAgent
