# Safety and Limitations

This kit uses local browser automation.

It does not:

- bypass platform security
- bypass login
- bypass 2FA
- bypass account checkpoints
- guarantee permanent compatibility
- guarantee compliance with every platform's policies
- remove the need for human approval

Stop immediately if you see:

- login screen
- 2FA
- checkpoint
- suspicious activity warning
- account locked
- identity confirmation
- unfamiliar security prompt

Recommended safety rules:

- start with one platform
- start with one post per day
- use human approval
- do not run high-volume posting
- do not use spammy content
- do not share account passwords
- avoid unknown cloud servers unless you understand the risks
